KMS激活教程

· 2022-06-10 10:48:40 ·

**kms地址：**kms.gxzyzd.com

**端口：**默认（1688）

**说明：**可用于激活 Office 批量版和 Windows 系统，联网自动续期。仅用于个人测试使用，公司电脑请勿使用，否则后果自负。

Windows

Windows 系统 kms 激活方法

以管理员身份运行命令提示符（cmd）

设置 kms 服务器地址

slmgr /skms kms.gxzyzd.com

激活命令

slmgr /ato

也可以直接使用二合一的命令

slmgr /skms kms.gxzyzd.com && slmgr /ato

说明

  1. 一般情况下下载的 VOL 版镜像都会内置密钥，直接设置 kms 地址并激活即可
  2. 查看系统版本的命令： wmic os get caption
  3. 如果内置 key 因其他原因而不可用，可以设置附录中的key 命令如下： slmgr /ipk xxxxx-xxxxx-xxxxx-xxxxx-xxxxx
  4. 一定要使用管理员身份运行 CMD 命令提示符，否则一定激活失败！！！
  5. 命令输入时请注意看好空格！
  6. 检查激活状态的命令： slmgr /dlv

Office 软件

Office 软件激活方法

以管理员身份打开命令提示符（cmd）

使用 cd 命令进入 Office 安装目录，例如：

Office2010 默认安装目录

32位的 cd C:\Program Files (x86)\Microsoft Office\Office14
64位的 cd C:\Program Files\Microsoft Office\Office14

Office2013 默认安装目录
32位的 cd C:\Program Files (x86)\Microsoft Office\Office15
64位的 cd C:\Program Files\Microsoft Office\Office15

Office2016默认安装目录
32位的 cd C:\Program Files (x86)\Microsoft Office\Office16
64位的 cd C:\Program Files\Microsoft Office\Office16

设置 kms 服务器

cscript ospp.vbs /sethst:kms.gxzyzd.com

激活命令

cscript ospp.vbs /act

也可以两条命令合二为一

cscript ospp.vbs /sethst:kms.gxzyzd.com && cscript ospp.vbs /act

说明

  1. 如果安装到了其他的盘，需要自行找到对应存放 OSPP.VBS 文件的路径。
  2. 一般下载安装 VOL 批量版本的应用都会默认一个 key 的，如果 key 不对可以在附录中找到对应的 key 在 Word 账户更改密钥。
  3. 一定要使用管理员身份运行 CMD 命令提示符，否则一定激活失败！！！
  4. 命令输入时请注意看好空格！
  5. 检查是否激活成功，打开 Word-文件-账户-不显示需要激活即可
